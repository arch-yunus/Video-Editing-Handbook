# Contributing Guidelines

First off, thank you for considering contributing to the Video Editing Handbook! It's people like you that make this a great resource.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and how to reproduce it.

### Suggesting Enhancements
- Open an issue to discuss your idea.
- Provide examples of why this enhancement would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added content, update the `README.md` or relevant module.
3. Ensure your markdown is well-formatted.
4. Submit your PR for review.

## Style Guide
- Use clear, professional Turkish for content.
- Keep technical terms in English where appropriate (e.g., *Color Grading*) but explain them.
- Use GitHub flavored markdown.
